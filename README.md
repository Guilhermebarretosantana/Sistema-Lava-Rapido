# Sistema-Lava-Rapido
Sistema de gestão financeira para lava rápido desenvolvido para registrar lavagens e gastos diários de forma pratica e organizada. O sistema permite cadastrar entradas e saídas, calcular automaticamente o lucro, gerar gráficos financeiros, exibir resumo por dia e relatório mensal, alem de salvar os dados automaticamente no navegador.
